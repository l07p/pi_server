# Dockerfile for Development Container
- Create the 'Dockerfile'
- Set Up the 'docker-compose.yml'
- Prepare the Directory Structure
 my-dev-container/
│
├── Dockerfile
├── docker-compose.yml
└── app/
    ├── todo-backend/
    │   └── src/
    │       └── main/
    │           └── java/
    │               └── com/
    │                   └── example/
    │                       └── todobackend/
    │                           └── TodoController.java
    └── todo-app/
        └── src/
            └── app/
                └── todo.service.ts

- Build and Run the Docker Container
- Access the Container and Use 'vim'
